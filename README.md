# LR1
Лабораторная работа №1
РАСЧЕТ ПАРАМЕТРОВ ГЕОМЕТРИЧЕСКИХ ФИГУР (C++)
Данная консольная программа на C++ предназначена для расчета параметров геометрических фигур.
Программа умеет вычислять периметр и площадь треугольника и прямоугольника, а также опознавать
равнобедренные треугольники и находить длину диагонали прямоугольника.

При запуске программы необходимо ввести номер вашей фигуры: 1 - прямоугольник, 2 - треугольник.
Далее программа запросит данные вашей фигуры - длины сторон.
Затем программа выведет параметры вашей фигуры.

При вводе неправильных (отрицательных) величин необходимо перезапустить программу. Также программа, при выборе треугольника, учитывает соотношения сторон: т.е. проверка сторон по неравенству треугольника.
